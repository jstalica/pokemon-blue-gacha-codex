# Pokemon Blue Gacha Client

This is the React frontend for the Pokemon Blue Gacha system built with Vite + React.

## Getting started

1. Install dependencies (`npm install`).
2. Start the dev server (`npm run dev`).
3. Open the provided URL (defaults to http://localhost:5173).

During development, API calls to `/api/*` proxy to `http://localhost:4000`; ensure the backend is running.
